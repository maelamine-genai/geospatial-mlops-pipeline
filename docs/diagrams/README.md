# Diagrams

This directory contains generated PNG diagrams and Mermaid source files for the repo documentation.

- `pipeline_lifecycle.png` — end-to-end stage flow.
- `core_plugin_architecture.png` — package and plugin boundary.
- `contracts_flow.png` — stage contracts and downstream consumers.
- `golden_evaluation.png` — full-scene sliding-window evaluation.
- `mlflow_promotion.png` — MLflow candidate/production lifecycle.
- `task_plugin_detail.png` — building task plugin internals.

The `.mmd` files are Mermaid sources that can be rendered by GitHub or documentation tools.
